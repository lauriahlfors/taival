# Taival

A travel itinerary planner application.

## About the project

This project is created by a small team of two, and our goal is to have a functional small-scale application that will showcase our skills and abilities technically and visually, with user experiense in mind.

We wanted to build this project using **React 18**, **TypeScript** and **Next.js 13** with its new `/app` directory and to incorporate features like **authentication** and **SQL databade** connection usign these technologies.

This projects UI and UX are designed in Figma with Tailwind CSS's style guides.

## Features

- User authentication
- Ability to create user generated content
